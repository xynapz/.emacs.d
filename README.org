Minimal Emacs config with Evil mode, LSP, and Org workflow.

* Setup

#+begin_src sh
git clone https://github.com/xynapz/.emacs.d ~/.emacs.d
#+end_src

First launch:
#+begin_src elisp
M-x xz/init-setup
#+end_src

Installs nerd-icons, tree-sitter grammars, PDF tools.

* Structure

| Path          | Purpose            |
|---------------+--------------------|

* Features

- *Evil Mode*: Full vim emulation with jk escape
- *LSP*: Eglot with pyright, clangd, typescript-language-server
- *Completion*: Vertico + Orderless + Corfu
- *Theme*: xz-nord (darker Nord variant)
- *Org*: Capture templates, agenda views
