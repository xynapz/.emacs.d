#+TITLE: emacs.d
#+AUTHOR: Angel Dhakal
#+OPTIONS: toc:nil num:nil

Robust, modular Emacs configuration with Evil mode, LSP, and Org workflow.

* Setup

#+begin_src sh
git clone https://github.com/xynapz/.emacs.d ~/.emacs.d
#+end_src

First launch:
1. Emacs will automatically bootstrap packages.
2. If prompted for VTerm compilation, answer *y* (requires =cmake=).
3. Install fonts and grammars:

#+begin_src elisp
M-x nerd-icons-install-fonts
M-x xz/install-treesit-grammars
#+end_src

* Structure

| Path             | Purpose            |
|------------------+--------------------|
| =init.el=        | Entry point        |
| =early-init.el=  | Startup optims     |
| =lisp/=          | Core Modules       |
| =themes/=        | xz-nord theme      |
| =snippets/=      | Yasnippet snippets |
| =docs/=          | Documentation      |

* Modules (lisp/)

| File               | Purpose                         |
|--------------------+---------------------------------|
| =xz-core.el=       | Settings (UTF-8, Desktop Fix)   |
| =xz-pkg.el=        | Package setup (ELPA/MELPA)      |
| =xz-ui.el=         | UI, Nord Theme, Doom Modeline   |
| =xz-editor.el=     | Evil, editing, windows          |
| =xz-completion.el= | Vertico, Consult, Corfu         |
| =xz-dev.el=        | LSP, Treesit, Magit, Projectile |
| =xz-org.el=        | Org mode, agenda, capture       |
| =xz-export.el=     | Blog export (ox-html)           |

* Features

- *Evil Mode*: Full vim emulation + evil-collection
- *LSP*: Robust lsp-mode with performance tuning
- *Completion*: Modern stack (Vertico + Consult + Corfu)
- *Theme*: xz-nord (darker Nord variant)
- *Org*: Pro capture templates and productivity system

* Requirements

| Package    | For       |
|------------+-----------|
| cmake      | vterm compilation |
| ripgrep    | Consult grep |
| fd         | Projectile |
| pyright    | Python LSP |
| clangd     | C/C++ LSP |
