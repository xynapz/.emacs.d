#+TITLE: Getting Started
#+OPTIONS: toc:nil

* Installation

1. Clone the repository:
   #+begin_src sh
   git clone https://github.com/xynapz/.emacs.d ~/.emacs.d
   #+end_src

2. Start Emacs. It will automatically download dependencies.

3. *Important*: If prompted to compile =vterm-module=, say *yes*. You will need =cmake= and =libtool= installed on your system.
   - Debian/Ubuntu: =sudo apt install cmake libtool-bin libvterm-dev=
   - Fedora: =sudo dnf install cmake libtool libvterm-devel=

4. Run post-install setup commands inside Emacs:
   - =M-x nerd-icons-install-fonts= (Installing fonts)
   - =M-x xz/install-treesit-grammars= (Installing tree-sitter languages)

* Verification

To verify everything is working:
1. Open a Python file (=test.py=).
2. Check if LSP starts (check mode line or =M-x lsp-describe-session=).
3. Try completion (start typing, see Corfu popup).
4. Try fuzzy finding files (=C-c p f= or =SPC f f= if mapped).

* Troubleshooting

** Icons are boxes
Run =M-x nerd-icons-install-fonts= and restart Emacs.

** VTerm error
Ensure you have CMake installed. Run =M-x vterm-module-compile=.

** LSP not Starting
Most likely the language server is missing.
- Python: =npm i -g pyright=
- JS/TS: =npm i -g typescript-language-server=
