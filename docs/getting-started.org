#+TITLE: Getting Started
#+OPTIONS: toc:nil

* Installation

1. Clone the repository:
   #+begin_src sh
   git clone https://github.com/xynapz/.emacs.d ~/.emacs.d
   #+end_src

2. Start Emacs. It will automatically download dependencies.

3. *Important*: If prompted to compile =vterm-module=, say *yes*. You need:
   - Debian/Ubuntu: =sudo apt install cmake libtool-bin libvterm-dev=
   - Fedora: =sudo dnf install cmake libtool libvterm-devel=

4. Run post-install setup inside Emacs:
   - =M-x nerd-icons-install-fonts= (Installing fonts)
   - =M-x xz/install-treesit-grammars= (Tree-sitter languages)

* Modal Editing (Meow)

This config uses *Meow* for modal editing (selection-first paradigm).

Quick start:
- =M-x meow-tutor= — Interactive 15-minute tutorial
- =SPC ?= — Cheatsheet

| State  | Key to Enter | Purpose          |
|--------+--------------+------------------|
| NORMAL | =ESC= / =jk= | Navigate/command |
| INSERT | =i= / =a=    | Type text        |

* Verification

To verify everything is working:
1. Open a Python file (=test.py=)
2. Check if LSP starts (mode line shows language server)
3. Try completion (start typing, see Corfu popup)
4. Try =SPC p f= for project file finding

* Troubleshooting

** Icons are boxes
Run =M-x nerd-icons-install-fonts= and restart Emacs.

** VTerm error
Ensure CMake is installed. Run =M-x vterm-module-compile=.

** LSP not Starting
Most likely the language server is missing.
- Python: =pip install python-lsp-server=
- JS/TS: =npm i -g typescript-language-server=
