* Overview

Custom variant of the [[https://www.nordtheme.com/][Nord theme]] with darker backgrounds.

Based on Nord by Sven Greb (MIT License).

* Changes from Default Nord

** Background Colors

Polar Night palette darkened using colors from doom-Iosvkem:

| Color | Original | xz-nord   | Usage     |
|-------+----------+-----------+-----------|
| nord0 | =#2E3440= | =#1b1d1e= | Main bg   |
| nord1 | =#3B4252= | =#262829= | hl-line   |
| nord2 | =#434C5E= | =#434C5E= | Selection |
| nord3 | =#4C566A= | =#4C566A= | Comments  |

** Comment Color

Brighter comments for visibility:

| Face    | Value     |
|---------+-----------|
| Comment | =#616e88= |

Default nord3 (=#4C566A=) was too dim.

* Installation

Theme lives in =themes/xz-nord-theme.el=.

Loaded via xz-editor.el:
#+begin_src elisp
(add-to-list 'custom-theme-load-path "~/.emacs.d/themes/")
(load-theme 'xz-nord t)
#+end_src

* Customizing

To modify colors, edit =themes/xz-nord-theme.el=.

Key sections:
- Line 16-36: Color definitions
- Line 41-97: Core/UI faces
- Line 99-121: Org mode faces
- Line 129-137: Magit faces
