* Global

| Key           | Action                    | Source         |
|---------------+---------------------------+----------------|
| =C-x C-b=     | Ibuffer                   | xz-editor      |
| =C-x b=       | consult-buffer            | xz-completion  |
| =C-x k=       | Kill this buffer          | xz-editor      |
| =C-x C-j=     | Dired jump                | xz-package     |
| =C-x g=       | Magit status              | xz-package     |
| =M-o=         | Other window              | xz-editor      |
| =C-c s=       | Save buffer               | xz-editor      |
| =C-c w=       | Delete trailing whitespace | xz-editor     |
| =C-c r=       | Revert buffer             | xz-editor      |
| =C-c r r=     | Clean restart             | xz-package     |
| =C-c ==       | Zoom in                   | xz-editor      |
| =C-c -=       | Zoom out                  | xz-editor      |
| =C-c /=       | Comment line              | xz-editor      |
| =C-c <left>=  | Winner undo               | xz-editor      |
| =C-c <right>= | Winner redo               | xz-editor      |
