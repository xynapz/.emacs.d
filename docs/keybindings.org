* Global

| Key           | Action                    | Source         |
|---------------+---------------------------+----------------|
| =C-x C-b=     | Ibuffer                   | xz-editor      |
| =C-x b=       | consult-buffer            | xz-completion  |
| =C-x k=       | Kill this buffer          | xz-editor      |
| =C-x g=       | Magit status              | xz-dev         |
| =C-c t=       | Toggle VTerm              | xz-dev         |
| =C-c r r=     | Clean restart             | xz-pkg         |
| =M-o=         | Other window              | xz-editor      |
| =C-c s=       | Save buffer               | xz-editor      |
| =C-c w=       | Delete trailing whitespace | xz-editor     |
| =C-c r=       | Revert buffer             | xz-editor      |
