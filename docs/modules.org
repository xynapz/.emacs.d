* Load Order

Managed in =init.el=:

#+begin_src elisp
(require 'xz-pkg)        ; Packages
(require 'xz-core)       ; Basics
(require 'xz-ui)         ; UI & Theme
(require 'xz-editor)     ; Customizations
(require 'xz-completion) ; Completion
(require 'xz-dev)        ; Dev Tools
(require 'xz-org)        ; Org
(require 'xz-export)     ; Export
#+end_src

* xz-core.el

Foundational settings.

| Setting           | Value               |
|-------------------+---------------------|
| Encoding          | UTF-8               |
| Backups           | Disabled            |
| Auto Revert       | Enabled (Global)    |
| Desktop           | Saved (filtered)    |
| Recentf           | Enabled (200 items) |

* xz-pkg.el

Package management infrastructure.

| Package             | Purpose           |
|---------------------+-------------------|
| use-package         | Config macro      |
| auto-package-update | Auto updates      |
| restart-emacs       | Safe restarting   |

* xz-ui.el

Visual appearance.

| Feature         | Implementation       |
|-----------------+----------------------|
| Theme           | xz-nord (Nord fork)  |
| Modeline        | doom-modeline        |
| Fonts           | Iosevka              |
| Icons           | nerd-icons           |

* xz-editor.el

Editing behavior and Evil integration.

| Package          | Purpose            |
|------------------+--------------------|
| evil             | Vim emulation      |
| evil-collection  | Omni-bindings      |
| evil-surround    | Surround ops       |
| evil-commentary  | Commenting         |
| display-line-numbers | Relative lines |
| elect-pair       | Auto brackets      |

* xz-completion.el

Modern completion stack.

| Package    | Purpose              |
|------------+----------------------|
| vertico    | Minibuffer UI        |
| orderless  | Fuzzy matching       |
| marginalia | Rich annotations     |
| consult    | Search commands      |
| corfu      | In-buffer completion |

* xz-dev.el

Developer tools and languages.

| Package     | Purpose                  |
|-------------+--------------------------|
| lsp-mode    | Language Server Protocol |
| treesit     | Syntax Highlighting      |
| projectile  | Project management       |
| magit       | Git client               |
| vterm       | Terminal                 |

** Languages Configured
- Python (pyright + treesit)
- JavaScript/TypeScript (ts-ls + treesit)
- C/C++ (clangd + treesit)
- Web (HTML/CSS LSPs)

* xz-org.el

Org mode setup. See [[file:org-guide.org][org-guide.org]].

* xz-export.el

Blog export for angeld site.
